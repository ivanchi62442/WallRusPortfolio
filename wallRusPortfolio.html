Live prices

Prices come from Yahoo Finance through your own free proxy — no API key, and it covers both US and European listings. Deploy the included worker.js to Cloudflare Workers (about 3 minutes, no card), then paste its URL below. It's stored only on this device.

On Buy transactions, use Yahoo tickers: AVUV (US), VWCE.DE (Xetra), VUAA.MI (Milan), BRK-B. Anything the proxy can't price, you can still set by hand in the Holdings table.

Base currency

All totals roll up into EUR. Foreign values are converted using ECB reference rates (via Frankfurter); currencies not covered, like RSD, use the EUR value you enter manually.
